### ⚠️ Riesgos
- **PDF corrupto o formato inesperado** → Solución: fallback a entrada manual de contenido
- **ElevenLabs rate limits** → Solución: batch processing con delays
- **OpenLMS versión antigua sin API** → Solución: import de SCORM packages
