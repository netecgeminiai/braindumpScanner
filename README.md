# Netec Gemini AI - Proyectos Automatizados

Este repositorio contiene los proyectos desarrollados y automatizados mediante IA.

## 📚 1. Curso ITIL Foundation v5 (`/itil-curso`)
Pipeline automatizado para generar aulas virtuales (videos HTML5/MP4 interactivos, guiones y audios con ElevenLabs).
- **`/planificacion`**: Mapas del curso y arquitectura.
- **`/assets`**: Imágenes oficiales, logos y diagramas (Spanish).
- **`/videos`**: Entregables finales (Audios MP3, Clips MP4 y HTML interactivo).

## 🛡️ 2. Braindump Scanner (`/`)
Herramienta de auditoría para detectar filtraciones de exámenes de certificación (ISTQB, etc.) en internet usando SerpApi y similitud semántica.
- `braindump_scanner_batch.py`: Procesador masivo de Excel.
- `braindump_scanner_pro.py`: Analizador inteligente.
- **`/entradas`**: Donde se colocan los Excels de exámenes a auditar.
- **`/resultados`**: Donde se generan los reportes de riesgo CSV.

## 🎥 3. Agencia Faceless - YouTube (`/youtube_faceless`)
Generador de contenido automatizado para canales de YouTube sin rostro (Nicho: Estoicismo/Psicología).
- `radar_estoicismo.py`: Buscador de tendencias y "outliers".
- `generar_voz.py`: Generador de locución premium.
